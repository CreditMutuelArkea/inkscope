InkscopeViz provides different visualizations of a Ceph cluster:
  • Ceph status: a ceph -s equivalent
  • OSDs status: all OSDs status with filters
  • OSDs map: OSDs hierarchical view
  • Pool usage: some pools statistics
  • Pools / pgs / osds: shows relationship between pools and osds
  • Crush map : a partial visualization of the Crush Map

Other visualizations:
  • Probe status: shows if probes are still active
  • Object lookup: shows on which OSD is or would be stored an object in a pool
